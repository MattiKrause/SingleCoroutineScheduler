scheduleSuspending

(JVM) suspend fun scheduleSuspending(task: Runnable): Disposable

Adds a task for the coroutine worker. If the buffer capacity is exceeded, this method suspends.

Parameters

task - The task that is to be executed

Return
A Disposable, which can be used to prevent execution of the task. Note that the canceled task is not removed from the buffer
